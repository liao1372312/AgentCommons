# Dataset

The full RapidAPI metadata and experiment task files are not included in this
repository snapshot because they are large and may contain third-party API
metadata. The public code can run the MCP service without the dataset.

For reproducing the paper experiments, place the prepared task file at:

```text
dataset/five_domain_1400_gold.jsonl
```

and keep the API metadata under:

```text
dataset/api/
```

These paths are ignored by Git by default.
